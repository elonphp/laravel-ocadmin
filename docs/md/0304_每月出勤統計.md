# 每月出勤統計

## 1. 設計目標

將每日出勤記錄彙整為月度摘要，提供薪資計算、績效考核、人力分析的基礎資料。每位員工每月產生一筆統計記錄。

### 1.1 資料流程

```
每日出勤統計（daily_attendances）── 30-31 筆/月/人
    ↓ 每月批次處理（每月 1 號自動 or 手動觸發）
每月出勤統計（monthly_summaries）── 1 筆/月/人
    ↓ 用於
薪資計算、績效考核、年度統計
```

## 2. 資料表設計

### monthly_summaries

```
monthly_summaries
├── id
├── employee_id              -- FK → employees, CASCADE
├── year_month               -- 年月 YYYY-MM（varchar 7）
│
├── scheduled_workdays       -- 應出勤天數（int, default 0）
├── actual_workdays          -- 實際出勤天數（int, default 0）
├── absent_days              -- 缺勤天數（int, default 0）
├── holiday_workdays         -- 假日上班天數（int, default 0）
│
├── scheduled_minutes        -- 應工作分鐘數（int, default 0）
├── work_minutes             -- 實際工作分鐘數（int, default 0）
├── overtime_minutes         -- 總加班分鐘數（int, default 0）
├── weekday_overtime_minutes -- 平日加班分鐘數（int, default 0）
├── holiday_overtime_minutes -- 假日加班分鐘數（int, default 0）
│
├── late_count               -- 遲到次數（int, default 0）
├── late_minutes             -- 遲到總分鐘數（int, default 0）
├── early_leave_count        -- 早退次數（int, default 0）
├── early_leave_minutes      -- 早退總分鐘數（int, default 0）
│
├── annual_leave_days        -- 特休天數（int, default 0）
├── sick_leave_days          -- 病假天數（int, default 0）
├── personal_leave_days      -- 事假天數（int, default 0）
├── other_leave_days         -- 其他假別天數（int, default 0）
│
├── status                   -- 狀態：draft / pending / approved / locked（見 §4）
├── note                     -- 備註（nullable, text）
├── reviewed_by              -- 審核人 ID（nullable, FK → users, nullOnDelete）
├── reviewed_at              -- 審核時間（nullable, timestamp）
├── calculated_at            -- 最後計算時間（nullable, timestamp）
│
├── timestamps
└── soft_deletes

UNIQUE(employee_id, year_month)
INDEX(year_month)
INDEX(status)
```

## 3. 欄位說明

### 3.1 天數統計

| 欄位 | 說明 | 來源 |
|------|------|------|
| scheduled_workdays | 應出勤天數 | 從行事曆（calendar_days）計算當月工作日數 |
| actual_workdays | 實際出勤天數 | `COUNT(is_absent = false AND work_minutes > 0)` |
| absent_days | 缺勤天數 | `COUNT(is_absent = true)` |
| holiday_workdays | 假日上班天數 | `COUNT(is_holiday_work = true)` |

### 3.2 工時統計

| 欄位 | 說明 | 計算方式 |
|------|------|----------|
| scheduled_minutes | 應工作分鐘數 | `SUM(scheduled_minutes)` |
| work_minutes | 實際工作分鐘數 | `SUM(work_minutes)` |
| overtime_minutes | 總加班分鐘數 | weekday_overtime + holiday_overtime |
| weekday_overtime_minutes | 平日加班 | 工作日超過標準工時的部分 |
| holiday_overtime_minutes | 假日加班 | 假日工作的全部時數 |

### 3.3 異常統計

| 欄位 | 說明 | 計算方式 |
|------|------|----------|
| late_count | 遲到次數 | `COUNT(is_late = true)` |
| late_minutes | 遲到總分鐘數 | `SUM(late_minutes)` |
| early_leave_count | 早退次數 | `COUNT(is_early_leave = true)` |
| early_leave_minutes | 早退總分鐘數 | `SUM(early_leave_minutes)` |

### 3.4 請假統計

請假系統（leaves）實作後才會有資料，目前預留欄位：

| 欄位 | 說明 |
|------|------|
| annual_leave_days | 特休天數 |
| sick_leave_days | 病假天數 |
| personal_leave_days | 事假天數 |
| other_leave_days | 其他假別天數 |

## 4. 狀態管理（status）

| 狀態 | 說明 | 時機 |
|------|------|------|
| `draft` | 草稿 | 月份尚未結束，持續更新中 |
| `pending` | 待審核 | 月份已結束，等待 HR 確認 |
| `approved` | 已審核 | HR 已審核確認 |
| `locked` | 已鎖定 | 已用於薪資計算，不可修改 |

**狀態流轉**：

```
draft（月份進行中，每日自動更新）
  ↓ 月份結束
pending（等待 HR 審核）
  ↓ HR 確認
approved（已審核）
  ↓ 薪資計算完成
locked（已鎖定，不可修改）
  ↓ 需特殊權限解鎖
approved（解鎖後可重新計算）
```

## 5. 核心業務邏輯

### 5.1 月統計計算流程

```
1. 取得或建立月統計記錄（employee_id + year_month 唯一）
2. 取得該月所有每日統計記錄（daily_attendances）
3. 從行事曆計算當月應出勤天數
4. 彙總各項統計數據（天數、工時、異常、請假）
5. 判斷狀態：
   - 當月尚未結束 → draft
   - 當月已結束且原為 draft → pending
   - 其他 → 保持現有狀態
6. 記錄 calculated_at 時間戳
```

### 5.2 排程

```
artisan 指令：attendance:process-monthly {year_month?}

預設處理上個月；可帶年月參數處理指定月份。

排程一：每月 1 號凌晨 02:00 處理上月（withoutOverlapping）
排程二：每天凌晨 03:00 更新當月草稿（可選）
```

## 6. 薪資計算整合

月統計提供薪資計算所需的完整資料：

| 月統計欄位 | 薪資計算用途 | 計算公式範例 |
|-----------|-------------|-------------|
| scheduled_minutes | 時薪基準 | 時薪 = 月薪 ÷ (scheduled_minutes ÷ 60) |
| work_minutes | 基本薪資 | — |
| weekday_overtime_minutes | 平日加班費 | 平日加班時數 × 時薪 × 1.33 |
| holiday_overtime_minutes | 假日加班費 | 假日加班時數 × 時薪 × 2.0 |
| absent_days | 缺勤扣款 | 缺勤天數 × 8 × 時薪 |
| late_minutes | 遲到扣款 | 遲到時數 × 時薪 |

### 鎖定與解鎖

- **鎖定**：薪資計算完成後，將月統計設為 `locked`，防止異動
- **解鎖**：需特殊權限，解鎖後退回 `approved` 狀態，並記錄解鎖原因

## 7. 績效指標

月統計可衍生以下績效指標：

| 指標 | 計算方式 |
|------|----------|
| 出勤率 | actual_workdays ÷ scheduled_workdays × 100% |
| 遲到頻率 | late_count ÷ actual_workdays × 100% |
| 早退頻率 | early_leave_count ÷ actual_workdays × 100% |
| 加班比例 | overtime_minutes ÷ work_minutes × 100% |

## 8. 與其他表的關聯

### 8.1 資料輸入

| 來源 | 提供資訊 |
|------|----------|
| daily_attendances（見 [0303](0303_每日出勤統計.md)） | 每日工時、遲到、早退、缺勤、加班等統計 |
| calendar_days（見 [0301](0301_行事曆管理.md)） | 計算當月應出勤天數（is_workday = true 的天數） |
| leaves（請假系統，待建立） | 統計各假別天數 |

### 8.2 資料輸出

| 目標 | 取用欄位 |
|------|----------|
| 薪資計算 | work_minutes、overtime 明細、absent_days、late_minutes |
| 績效考核 | 出勤率、遲到/早退次數 |
| 年度統計報表 | 12 個月彙總 |

## 9. 實作優先序

### Phase 1

- 資料表建立
- 基本統計邏輯（從 daily_attendances 彙總）
- 批次指令

### Phase 2

- 薪資計算整合
- 鎖定 / 解鎖機制
- 審核流程

### Phase 3

- 報表匯出（Excel）
- 年度統計分析
- 績效考核整合
