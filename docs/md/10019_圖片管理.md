# 0114 圖片管理（Image Manager）

## 概述

圖片管理器是後台共用元件，提供檔案瀏覽、上傳、刪除、建立資料夾等功能。
使用者在表單中點擊「選擇圖片」按鈕後，彈出 Bootstrap Modal 檔案管理器，選取圖片後回填路徑到表單隱藏欄位並更新縮圖預覽。

此元件放置於 Core 層級，屬於系統層級的跨專案通用功能。

### 來源與改良

本系統的 `ImageLibrary` 參考自 OpenCart 4.x（`system/library/image.php`），並加以改良：

- **移除未使用方法**：OpenCart 原版包含 `crop()`、`watermark()`、`rotate()`、`filter()`、`text()`、`merge()`、`html2rgb()` 等方法，但在 OpenCart 4.x 自身原始碼中皆無任何呼叫，屬於死碼。本系統僅保留實際使用的 `resize()` 與 `save()`。
- **PNG gAMA 修復**：新增 `cleanImage()` 邏輯，偵測 PNG 的 gAMA chunk 並重建像素，解決跨瀏覽器色彩異常問題。
- **透明背景處理**：`save()` 針對 PNG / WEBP 格式加入 `imagealphablending` + `imagesavealpha` 設定，確保透明背景正確儲存。
- **PHP 8.0+ 適配**：以 `unset()` 取代已 deprecated 的 `imagedestroy()`。

## Storage 架構

透過 `php artisan storage:link` 建立 symlink：`public/storage` → `storage/app/public`

- Windows 上 Laravel 使用 `mklink /D`，需以**系統管理員**身份執行 terminal
- `config/filesystems.php` 已正確設定，無需修改

```
storage/app/public/
├── image/
│   ├── catalog/          ← 商品圖片
│   │   └── SZ25.jpg
│   └── no_image.png      ← 預設無圖片佔位圖
└── cache/                ← 縮圖快取（自動產生）
    └── image/
        └── catalog/
            └── SZ25-100x100.jpg
```

**公開 URL 範例：**

| 用途 | URL |
|------|-----|
| 原圖 | `https://domain/storage/image/catalog/SZ25.jpg` |
| 縮圖 | `https://domain/storage/cache/image/catalog/SZ25-100x100.jpg` |

> `public/assets/ocadmin/image/no_image.png` 作為表單預覽的 placeholder。
> `storage/app/public/image/no_image.png` 供 ImageLibrary PNG 處理 fallback 使用。

## 核心元件

### ImageLibrary

**檔案：** `app/Libraries/ImageLibrary.php`

以 PHP GD 擴充處理圖片，支援 PNG / JPEG / GIF / WEBP 格式。
提供 `resize()` 與 `save()` 方法，包含 PNG gAMA chunk 偵測與重建邏輯（`cleanImage()`）。

### ImageHelper

**檔案：** `app/Helpers/Classes/ImageHelper.php`

靜態方法 `resize()` 負責產生縮圖快取：

- 接收相對路徑（如 `image/catalog/SZ25.jpg`）與目標尺寸
- 快取路徑格式：`cache/{原路徑}-{W}x{H}.{ext}`
- 快取存在且較新時直接回傳 URL，否則透過 ImageLibrary 產生縮圖
- 使用 `realpath()` 進行路徑安全檢查，防止目錄遍歷

```php
$thumbUrl = ImageHelper::resize('image/catalog/SZ25.jpg', 100, 100);
// → https://domain/storage/cache/image/catalog/SZ25-100x100.jpg
```

### ImageManagerController

**檔案：** `app/Portals/Ocadmin/Core/Controllers/Common/ImageManagerController.php`

| 方法 | HTTP | 路由 | 說明 |
|------|------|------|------|
| `index()` | GET | `/common/image-manager` | 回傳 Modal 外殼 HTML（帶 target/thumb 參數） |
| `list()` | GET | `/common/image-manager/list` | 回傳檔案瀏覽器 partial（目錄/檔案列表、分頁） |
| `upload()` | POST | `/common/image-manager/upload` | 檔案上傳（JSON 回應） |
| `folder()` | POST | `/common/image-manager/folder` | 建立資料夾（JSON 回應） |
| `delete()` | POST | `/common/image-manager/delete` | 刪除檔案/資料夾（JSON 回應） |

**核心邏輯：**

- 檔案根目錄：`storage_path('app/public/image/')`
- `list()`：glob 掃描目錄 + 檔案，支援 filter_name / sort / order / page，每頁 16 筆
- 縮圖透過 `ImageHelper::resize()` 產生
- `upload()`：驗證副檔名（ico/jpg/jpeg/png/gif/webp）、MIME 類型、檔名長度（4–255）、上傳大小
- `folder()`：在 `image/catalog/` 下建立子目錄，名稱長度需 3–128 字元
- `delete()`：遞迴刪除，安全檢查路徑不超出 storage 範圍

## Views

### `common/imgmanager/index.blade.php`

Bootstrap 5 Modal（`#modal-image`），載入後立即 AJAX 呼叫 `list` 路由填充 `.modal-body`。

JS 事件處理：
- 導航（parent / refresh）、搜尋、上傳、建立資料夾、刪除
- 圖片選取回填、分頁、排序
- Grid / List 雙視圖切換，模式透過 `localStorage` 持久化
- 欄位表頭排序（名稱、大小、日期），支援 asc / desc 切換

圖片選取回填邏輯（同時處理 Grid view 及 List view 的 DOM 結構差異）：

```javascript
$('{{ $thumb }}').attr('src', imgSrc);     // 更新預覽
$('{{ $target }}').val(pathVal);            // 寫入路徑
```

### `common/imgmanager/list.blade.php`

工具列：上一層、重新整理、上傳、新增資料夾、刪除、搜尋

- **Grid view**：網格佈局 `row-cols-sm-3 row-cols-lg-4`，目錄以資料夾圖示、圖片以縮圖顯示
- **List view**：表格佈局，欄位為 checkbox / 預覽 / 檔名 / 大小 / 修改日期，表頭可點擊排序
- 視圖切換按鈕（小圖 / 列表），透過 `localStorage('imgmanager_view')` 記憶
- 每項有 checkbox 用於批次刪除
- 底部分頁

## 前端觸發機制

**檔案：** `public/assets/ocadmin/javascript/common.js`

監聽 `[data-oc-toggle='image']` 點擊事件，從 `#imgmanager-url` hidden input 取得圖片管理器 URL，以 AJAX 載入 Modal：

```javascript
$(document).on('click', '[data-oc-toggle=\'image\']', function (e) {
    var element = this;
    $('#modal-image').remove();

    var imgmanagerUrl = $('#imgmanager-url').val();

    $.ajax({
        url: imgmanagerUrl + '?target=' + encodeURIComponent($(element).attr('data-oc-target'))
             + '&thumb=' + encodeURIComponent($(element).attr('data-oc-thumb')),
        dataType: 'html',
        success: function (html) {
            $('body').append(html);
            $('#modal-image').modal('show');
        }
    });
});
```

使用圖片選擇器的表單頁面需加入 hidden input：

```html
<input type="hidden" id="imgmanager-url" value="{{ route('lang.ocadmin.common.image-manager.index') }}">
```

## 表單整合範例：Option 選項值圖片

### Controller（`OptionController.php`）

`edit()` 方法為每個 option_value 產生縮圖 URL：

```php
$data['defaultThumb'] = asset('assets/ocadmin/image/no_image.png');

foreach ($data['optionValues'] as $value) {
    if ($value->image && is_file(storage_path('app/public/' . $value->image))) {
        $value->thumb = ImageHelper::resize($value->image, 100, 100);
    } else {
        $value->thumb = $data['defaultThumb'];
    }
}
```

`saveOptionValues()` 方法儲存圖片路徑：

```php
$optionValue = $option->optionValues()->create([
    'code' => $valueData['code'] ?? null,
    'image' => $valueData['image'] ?? null,
    'sort_order' => $valueData['sort_order'] ?? $index,
]);
```

驗證規則：`'option_value.*.image' => 'nullable|string|max:255'`

### View（`form.blade.php`）

每列 option_value 包含圖片欄位：

```blade
<td class="text-center">
    <div class="image">
        <img src="{{ $value->thumb }}"
             id="thumb-image-{{ $index }}"
             data-oc-placeholder="{{ asset('assets/ocadmin/image/no_image.png') }}"
             class="img-thumbnail" style="max-width:100px;">
        <input type="hidden" name="option_value[{{ $index }}][image]"
               value="{{ $value->image }}" id="input-image-{{ $index }}">
        <div class="btn-group mt-1">
            <button type="button" data-oc-toggle="image"
                    data-oc-target="#input-image-{{ $index }}"
                    data-oc-thumb="#thumb-image-{{ $index }}"
                    class="btn btn-primary btn-sm"><i class="fa-solid fa-pencil"></i></button>
            <button type="button" data-oc-toggle="clear"
                    data-oc-target="#input-image-{{ $index }}"
                    data-oc-thumb="#thumb-image-{{ $index }}"
                    class="btn btn-warning btn-sm"><i class="fa-solid fa-trash-can"></i></button>
        </div>
    </div>
</td>
```

`addOptionValue()` JS 函式中同步包含相同的圖片欄位 HTML。

## 資料流程

```
使用者點擊「Edit」按鈕
    │  data-oc-toggle="image"
    ▼
common.js 攔截 → AJAX GET /common/image-manager?target=...&thumb=...
    │
    ▼
ImageManagerController::index() → 回傳 Modal HTML (imgmanager/index.blade.php)
    │  append to body, show modal
    ▼
Modal JS 自動觸發 → AJAX GET /common/image-manager/list
    │
    ▼
ImageManagerController::list() → glob 掃描 storage/app/public/image/
    │  每張圖片產生 thumb URL via ImageHelper::resize()
    ▼
回傳 imgmanager/list.blade.php → 顯示目錄/圖片網格
    │
    ▼ 使用者點擊圖片縮圖 (a.thumbnail)
    │
$('#input-image-N').val('image/catalog/SZ25.jpg')     ← 寫入隱藏 input
$('#thumb-image-N').attr('src', '..thumbnail URL..')   ← 更新預覽圖
    │
    ▼ 使用者儲存表單
    │
OptionController::store/update() → saveOptionValues()
    │
    ▼
option_values.image = 'image/catalog/SZ25.jpg'         ← 存入 DB
```
