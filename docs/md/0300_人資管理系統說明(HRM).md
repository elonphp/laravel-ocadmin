# 系統說明

本系統包含以下兩大模組，各自獨立前綴，可分別發展：

## 1. 人資管理（HRM）

資料表前綴：`hrm_`

- **出勤打卡**：員工每日上下班打卡記錄與管理
- **薪資計算**：依據出勤、加班、請假等資料進行薪資結算
- **假日管理**：國定假日、特休、請假類別與額度管理
- **排班管理**：班別設定、員工排班與調班作業

## 2. 資產管理（Asset）

資料表前綴：`asset_`

- **資產類別**：資產分類與管理
- **資產項目**：公司資產登錄、狀態追蹤
- **資產配發**：資產領用、歸還、調撥記錄

資產如需關聯員工，透過 `asset_assignments.employee_id` 與 HRM 模組串接。
