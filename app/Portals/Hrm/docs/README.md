## 🎯 快速開始

### 新人上手流程

1. **了解系統** → 閱讀 [1000_差勤系統概述.md](specs/1000_差勤系統概述.md)
2. **查看 API** → 閱讀 [API.md](guides/API.md)
3. **實作功能** → 根據 specs/ 中的規格文件開發
4. **測試驗證** → 使用 API.md 和 COMMANDS.md 測試

### 前端工程師

1. 閱讀 [../README.md](../README.md) - HRM Portal 總覽
2. 閱讀 [API.md](guides/API.md) - 了解後端 API
3. 閱讀 specs/ - 了解業務邏輯

### 後端工程師

1. 閱讀 specs/ - 了解業務需求
2. 閱讀 [COMMANDS.md](guides/COMMANDS.md) - 了解 Console 命令
3. 實作 Controller 和 Service
