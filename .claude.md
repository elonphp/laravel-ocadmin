# laravel-ocadmin

借用 OpenCart 後台前端的 Laravel 後台管理系統。

## 核心特點

- 基於 Laravel 框架
- 前端借用 OpenCart 後台樣式
- Controller 設計參考 OpenCart 後台架構

## 技術棧

- 後端：PHP + Laravel
- 資料庫：MariaDB / MySQL
- 前端：OpenCart Admin 樣式 + Bootstrap

## 開發環境

當前作業系統環境使用多版本 PHP。

**Claude Code 執行時請使用完整路徑：**
```bash
# PHP
D:/Servers/PHP/php84/php.exe D:/Codes/PHP/Dboem/sunline.test/htdocsNew/laravel/artisan ...

# Composer
D:/Servers/PHP/php84/php.exe D:/Servers/PHP/php84/composer.phar ...
```

**使用者手動執行可用批次檔：**
```bash
.\php.bat artisan ...
.\composer.bat ...
```

## 專案結構

- `portals/Ocadmin` - 後台管理模組
  - `app/Http/Controllers` - 控制器
  - `resources/views` - 視圖模板
  - `routes/web.php` - 路由定義
